# Task

Consider yourself a Senior Go developer.
I am writing a configuration app. A single configuration is called a Tegn, and it can be installed.
I would like to create the ability to uninstall it. 

For each code file below that contains `ExecInstall`, implement `ExecUninstall`.
For configuration blocks (e.g., `# <BEGIN> something`, `# <END> something`), use regex to remove them.
For created files with the header `DO NOT EDIT` (such as `local-...` files), remove them.
DO NOT remove `.zshrc`, `.bashrc`, or similar files that may have contained user data before (or after) the installation.
DO NOT uninstall applications (do not touch `genericpackage` or `genericdownload`).
Your output MUST use the following format: an h2 markdown heading for the file name (the file path to be created), and the markdown code block containing the file content.
