# Task

Consider yourself a Senior Go developer.
I would like to use the `tview` library to create a Terminal User Interface for my application.

Application purpose: PC configuration (zsh config presets, app installation, etc.).
UI purpose: 
- To show available Tegns and Tegnsetts.
- To select Tegns and Tegnsetts.
- To search for Tegns and Tegnsetts by their ID.
- To view descriptions of Tegns and Tegnsetts.

UI usage pipeline:
1) First, the user is shown a UI with the Tegnsett selector. The user can enable/disable the Tegnsett (using Space) and enter the Tegnsett (using Enter) -> (2).
2) The Tegnsett UI contains a list of the Tegns within that Tegnsett. The user can enable/disable the Tegn (using Space) and enter the Tegn (using Enter, only if it has properties) -> (3).
3) The property UI contains a list of the properties.

At the bottom, there should be buttons: Help, Confirm, Search, Exit.

There should also be shortcuts:
- [F10] or [Ctrl-C] to exit (Exit button). Before exiting, an Exit Confirmation modal window should appear.
- [F5] or [Ctrl-X] to confirm the selection (Confirm button). A Confirmation modal should appear.
- [F1] or [?] to show "Help" — a modal with the Tegn, Tegnsett, or parameter description should appear.
- [F3] or [/] to search — a modal with an input box should appear. The user enters the "id" of a Tegn or Tegnsett, and the UI should open it.

Your output MUST use the following format: an h2 markdown heading for the file name (the file path to be created), and the markdown code block containing the file content.
